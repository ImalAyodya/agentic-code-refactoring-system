# Agentic Intelligent Code Refactoring Assistant

This repository contains a research prototype for an intelligent
agent-based system that automatically analyzes legacy codebases,
plans refactoring actions, safely applies transformations, and
supports human-in-the-loop interaction.

## Key Features
- Agent-based program understanding
- Intelligent refactoring planning
- Safe code transformation with validation
- Developer-in-the-loop orchestration

## Technology Stack
- Python
- AST-based code analysis
- Multi-agent architecture
- GitHub repository-level refactoring
